Hospedagem

O que é Hospedagem?

A hospedagem é o espaço ou um lugar onde uma pessoa pode ficar por um determinado tempo.

Diferença entre Domínio e Hospedagem

  • Hospedagem é a casa em si, onde ficam os móveis.
  • Domínio é o lugar, quarterão ou o bairro onde a pessoa fica.(endereço)

Como Publicar um Site Simples

  1. Registrar um domínio.
  2. Contratar uma hospedagem.
  3. Enviar os arquivos HTML, CSS e JavaScript para o servidor.
  4. Pronto! Seu site estará acessível pelo domínio.

Tipos de Hospedagem

  • Hospedagem Compartilhada: vários sites dividem o mesmo servidor.
  • Hospedagem VPS: cada site tem uma parte dedicada do servidor.
